# Contributing

Thanks for your interest in contributing to Excel Bot!

## Development setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
python -m pip install -e .
python -m pip install pytest
```

3. Run tests:

```bash
python -m pytest -v
```

## Pull requests

- Keep changes focused and clearly described.
- Update tests if you change behavior.
- If you add new functionality, consider updating the README or CHANGELOG.
